# Capsula - Moodle Activity Plugin

Capsula is a secure hybrid viewer for Moodle that allows instructors to display Videos and PDFs with advanced security and usability features.

## Features

- **Hybrid Display**: Show Video only, PDF only, or both side-by-side.
- **Secure PDF Viewer**:
  - Prevents downloading, printing, and context menus.
  - Dynamic watermarking with user details (Name, Email, Time, IP).
  - Modern dark-themed UI similar to Google Drive.
  - Custom zoom controls.
- **Secure Video Player**:
  - 16:9 Aspect Ratio.
  - clean interface.
- **Direct URL Support**: Load content via Google Drive URLs or direct links.

## Installation

1. Clone or extract this repository into `mod/capsula` in your Moodle installation.
2. Log in to your Moodle site as an administrator.
3. Follow the prompts to upgrade the database.

## Usage

1. Go to a course and turn editing on.
2. Click "Add an activity or resource".
3. Select "Capsula".
4. Configure the settings:
   - **Name**: Activity name.
   - **Mode**: Choose between Video & PDF, Video only, or PDF only.
   - **Video URL**: Direct link to the MP4 or stream.
   - **PDF URL**: Direct link to the PDF document.
5. Save and display.
